# Logseq Social Preview Studio

Create fast content previews for major social platforms and ad formats directly in Logseq.

## What it does

This plugin generates structured preview cards for:

- Instagram post
- Snapchat story
- Facebook feed post
- YouTube video
- TikTok short
- AdSense-style display ad
- Generic social card (for any other channel)

Each generated preview includes:

- Platform metadata
- Hook and primary text draft
- CTA suggestion
- Visual direction notes
- A short QA checklist before publishing

## Installation

1. Open Logseq
2. Go to Plugins -> Load unpacked plugin
3. Select this folder:

plugins/logseq-social-preview-studio/plugin

4. Start using slash commands

## Commands

- Social Preview Studio: Open Dashboard
- Social Preview Studio: New Preview (Choose Platform)
- Social Preview Studio: Instagram Post
- Social Preview Studio: Snapchat Story
- Social Preview Studio: Facebook Feed
- Social Preview Studio: YouTube Video
- Social Preview Studio: TikTok Short
- Social Preview Studio: AdSense Display Ad
- Social Preview Studio: Generic Social Card

## Settings

- Target Page Name
- Creator or Brand Name
- Default Handle
- Default CTA

## Typical workflow

1. Run Social Preview Studio: Open Dashboard
2. Run Social Preview Studio: New Preview (Choose Platform)
3. Enter your campaign brief
4. Review generated post draft and checklist
5. Duplicate and tweak for each platform
